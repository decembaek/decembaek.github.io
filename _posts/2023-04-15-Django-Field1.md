# Django 장고 DB 필드 정수값 넣기 



```python
class modelName(models.Model):
	Field_name = models.PositiveIntegerField()
```

**PositiveIntegerField는** 파이썬에서 **int** 인스턴스로 표시되는 정수이다. 

이 필드는 IntegerField 비슷하지만 양수 또는 0(0)이어야 합니
